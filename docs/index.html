Claude Code on your server,
from Telegram

Full bash, file operations, and search on any machine you own. Runs on your existing Claude Code plan — no extra API costs. Set up in 2 minutes.

Quick Start
git clone https://github.com/cogent42/cogent42.github.io.git
cd cogent42
node setup.js  # interactive — walks you through everything

Requires Node.js 18+, a Telegram bot token, and a Claude Code subscription

What you get
>_

Full server access

Bash execution, file read/write/edit, code search. Everything Claude Code can do, from your phone.

🧠

Persistent memory

Sessions survive restarts. Facts and decisions are extracted into a knowledge base that carries across conversations.

💳

Uses your existing plan

Runs on your Claude Code subscription. No separate API key or pay-per-token billing. Just authenticate claude on your server and go.

🔒

Single-user lock

Locked to your Telegram user ID. Other users are silently rejected. Your server, your bot.

📎

Photos & documents

Send images and files via Telegram. Claude processes them on your server — screenshot debugging, document analysis, anything.

⇅

Smart model routing

Defaults to Sonnet 4.6 for speed. Auto-escalates to Opus 4.6 when a query needs more. Switch manually with /opus and /sonnet.

How it works
1

Get your tokens

Create a bot via @BotFather on Telegram. Have your Claude Code subscription ready.

2

Run the setup

node setup.js installs the Claude Code CLI if needed, authenticates with your subscription, configures the bot, and optionally sets up PM2.

3

Message your bot

Open Telegram, send a message. Claude Code executes on your server and replies with the result.

Commands
/startWelcome message and command reference
/resetExtract knowledge, archive session, start fresh
/cancelAbort the current in-flight query
/statusUptime, disk, memory, model, and session info
/historySession statistics
/opusSwitch to Claude Opus 4.6
/sonnetSwitch to Claude Sonnet 4.6
/knowledgeView all stored knowledge entries
Configuration
TELEGRAM_BOT_TOKEN required
Bot token from @BotFather
TELEGRAM_USER_ID required
Your Telegram user ID from @userinfobot
MAX_TURNS optional
Max agentic turns per query. Default: 25
WORKING_DIRECTORY optional
Directory where Claude operates. Default: ~
BOT_NAME optional
Display name for your bot. Default: cogent42
BOT_PERSONALITY optional
Optional personality prompt (e.g. "concise and direct")
Security note

Cogent runs with bypassPermissions mode, giving Claude unrestricted access to your server — bash, filesystem, everything. Only run this on machines you're comfortable giving full access to. The bot is locked to a single Telegram user ID; unauthorized users are silently rejected.
